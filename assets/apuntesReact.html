¿Qué es ReactJs?

React es una biblioteca de JavaScript para poder desarrollar interfaces de usuario.
Esta biblioteca nos permite crear aplicaciones web interactivas que responden velozmente como si fueran aplicaciones nativas.
React es una herramienta de JavaScript open source que fue creada por facebook.

NodeJs:

Para utilizar React hay que descargar NodeJs.
NodeJs es prácticamente un programa que nos permite ejecutar JavaScript fuera del ordenador.

¿Cómo crear un nuevo proyecto de ReactJs?

Una de las herramientas más comunes para crear nuevos proyectos en react es "Create react app". Otra es "Vite".
Abrir la consola en donde queremos que se cree la carpeta.
Con create react app: npx create-react-app nombre-del-proyecto (El nombre no puede tener mayúsculas).

¿Cómo iniciar nuestro proyecto?

Ir al CMD de la carpeta (Se puede cambiar dentro de la consola utilizando: cd seguido del nombre de la carpeta)
Una vez dentro ejecutar npm start.

Comandos:

npm start: Es para iniciar nuestro proyecto. (Ejecuta un servidor de desarrollo)
npm run build: Nos permite construir la aplicación (o mejor dicho prepararla para producción). Esto lo utilizaremos cuando ya hayamos terminado nuestro proyecto.
npm test: Este comando se utiliza para hacer testing.
npm run eject: Permite modificar las configuraciones de create react app.

Carpetas y archivos:

src: Es donde pondremos nuestro codigo, es donde vamos a crear archivos. (En ReactJs se trabaja con JS, en él se generarán las interfaces de HTML)
public: Dentro habrá un HTML que es en el cual se subirá toda la información cargada de nuestro JS. (Tambien habrá unos archivos png, ico, estos podemos eliminarlos ya que son de ejemplo)
build: Esta carpeta es creada despues de terminar el proyecto. (ejecutando npm run build en consola). Esta carpeta contiene nuestro codigo para ser subido a produccion, es decir, solo se subiria esta carpeta.
node_modules: Hay carpetas con archivos para que react o la propia biblioteca pueda funcionar (No deberiamos tocar nada de esto)

Importar biblioteca:

Para poder empezar a utilizar React primero debemos importar la biblioteca, cuando creamos nuestro proyecto, react ya instaló un paquete llamado "react" entonces hay que requerirlo.
Despues de eso hay que importar otra biblioteca adicional la cual es para que react sepa que vamos a trabajar en un proyecto web.
Luego tenemos que decirle a react donde vamos a montar la aplicacion (En este caso hay un div con id "Root")

import React from "react" // Importa la biblioteca de React.
import ReactDom from "react-dom/client" // Importa la biblioteca dom de react.

const root = ReactDom.createRoot(document.getElementById("root")) // Selecciona el div donde irá la información.
root.render( ) // Acá se pone qué queremos poner dentro de nuestra aplicación.
